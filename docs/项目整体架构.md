# 简介
cm是一个微服务项目。
# 项目说明
| 项目           | 说明                     |
| ------------ | ---------------------- |
| cm-framework | SpringCloudAlibaba基础框架 |
| cm-gateway   | Gateway网关              |
| cm-auth      | 认证中心                   |
|              |                        |
# 目录介绍
- docs: 项目文档
- backend: 后端代码
	- cm-framework: SpringCloudAlibaba基础框架
		- cm-dependencies: 基础依赖
		- cm-web-spring-boot-starter: web服务starter
	- cm-gateway: gateway网关
	- cm-auth: 认证中心
- frontend: 前端代码
- migration: 版本变更记录
